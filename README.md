# 每日诗词桌面小程序
这是一个基于 Python 和 Tkinter 开发的桌面诗词小组件，用于在桌面上持续展示每日诗句。它支持自动刷新、透明背景、主题与样式自定义，并提供点击穿透与开机自启等实用功能，在不打扰使用的前提下为桌面增加一点文学氛围。

---

# 📜 Daily Poetry Widget

一个简单的桌面小工具，每天在屏幕角落给你一首诗。

不打扰、不占地方，但偶尔抬头能看到一句还不错的句子。

---


## ✨ 有什么功能

* 桌面悬浮（一直在最上层）
* 自动获取诗词（默认每小时更新一次）
* 支持拖动位置
* 右键菜单操作
* 深色 / 浅色主题
* 字体大小、颜色可调
* 背景透明度可调
* 🔒 锁定后可点击穿透（不挡鼠标）
* 支持开机自启
* 配置自动保存

---

## 📦 怎么用（推荐）

已经打包好了 exe，直接用就行。

1. 下载 `DailyPoetryWidget.exe`
2. 双击运行
3. 右键打开菜单，自己慢慢调

基本不需要折腾环境。

---

## 🖱️ 一些操作说明

* **拖动**：左键按住移动
* **右键**：打开菜单
* **锁定界面**：开启后可以“点穿”（类似壁纸）
* **右上角小锁**：解锁用

---

## ⚙️ 设置里可以改什么

* 背景透明度
* 字体大小（诗句 / 作者）
* 显示内容（只显示诗 / 带作者 / 带标题）
* 颜色（背景、文字）
* 主题（深色 / 浅色）

改完点保存就行，会写进本地配置文件。
